# rpilocator bot

This script just returns RSS feed from `rpilocator.com` in a JSON readable format.

## Requirements

- [NPM](https://www.npmjs.com) / [Yarn](https://yarnpkg.com)

## Installation

Install deps with Yarn then run the script.

```
yarn
yarn start
```

## Commands
```
yarn start                 # Start script
```
